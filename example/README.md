# example

- The source mateiral was download from [青空文庫](https://www.aozora.gr.jp/cards/000148/card799.html) and converted to ePub with [AozoraEpub3](https://github.com/kyukyunyorituryo/AozoraEpub3/releases). 
- The chapterized txts were converted with `nk example/\[夏目漱石\]\ 夢十夜.epub`.
- The single-file txt was converted with `nk example/\[夏目漱石\]\ 夢十夜.epub --single-file`
- To convert the chapterized txts to mp3s, run `nk tts example/\[夏目漱石\]\ 夢十夜 --speaker 13`
