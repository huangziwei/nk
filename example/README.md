# example

- The source mateiral was download from [青空文庫](https://www.aozora.gr.jp/cards/000125/card1317.html) and converted to .epub with [AozoraEpub3](https://github.com/kyukyunyorituryo/AozoraEpub3/releases). 
- The chapterized txts were converted with `nk example/\[小栗虫太郎\]\ 黒死館殺人事件.epub`.
- The single-file txt was converted with `nk example/\[小栗虫太郎\]\ 黒死館殺人事件.epub --single-file`
- To convert the chapterized txts to mp3s, run `nk tts example/\[小栗虫太郎\]\ 黒死館殺人事件 --speaker 13`
